# 选择困难症终结者需求文档

## 1. 网站概述

### 1.1 网站名称
选择困难症终结者

### 1.2 网站描述
一个通过AI智能分析和多种趣味化选择工具帮助用户解决选择困难的决策辅助网站。用户可以通过文字或语音输入选择问题，AI会自动分析并提供最适合的决策方式。

### 1.3 网站功能
- 智能选择助手：AI分析用户选择问题，自动匹配合适的功能工具
- 掷硬币：为正反面指定事件，随机选择结果
- 掷色子：语音或文字输入事件，系统识别并随机选择
- 概率转盘：自定义条目和概率，生成转盘进行选择
- AI分析：智能分析各种情况，提供选择建议
- 答案之书：随机显示指导性语句帮助决策

## 2. 功能详细说明

### 2.1 首页重构
- 顶部展示网站Title和简要文字介绍
- 隐藏五个功能模块的具体入口区域
- 底部增加输入框，提示文字：'请输入或说出您想要做出的选择'（支持文字输入和语音输入）
- 添加'确认'按钮供用户提交输入内容

### 2.2 智能选择助手逻辑
- 用户在首页输入选择问题后，AI自动分析问题类型和复杂程度
- 根据分析结果自动选择最适合的功能工具（掷硬币、掷色子、概率转盘、AI分析、答案之书）
- 自动填充所选功能工具中的相关参数和内容
- 直接跳转到对应功能页面，展示完整的决策界面

### 2.3 功能页面通用改造
- 在所有功能页面右下角增加'消息'图标
- 点击'消息'图标可向左侧展开AI聊天对话框
- 聊天框支持文字输入和语音输入
- AI功能：
  - 首先分析用户输入是否需要切换到其他功能工具
  - 如果不需要切换，则分析用户是否想修改当前功能的具体参数
  - 根据分析结果动态调整页面内容或执行相应操作

### 2.4 各功能模块详细说明

#### 2.4.1 掷硬币模式
- 用户输入：AI自动识别选择问题并为硬币正反面分配事件
- 界面展示：硬币图形、事件描述、'掷硬币'按钮
- 交互：点击'掷硬币'执行随机选择，显示翻转动画和结果

#### 2.4.2 掷色子模式
- 用户输入：AI识别选择问题并为色子六个面分配事件
- 界面展示：色子图形、事件列表、'摇色子'按钮
- 交互：点击'摇色子'显示滚动动画和最终结果

#### 2.4.3 概率转盘模式
- 用户输入：AI分析选择问题并识别不同选项及其相对概率
- 界面展示：可视化概率转盘、选项列表
- 交互：点击'开始转盘'执行旋转动画并显示选择结果

#### 2.4.4 AI分析模式
- 用户输入：AI详细分析选择问题，识别各个选项的优劣势
- 界面展示：分析结果、选项对比、建议结论
- 交互：提供'重新分析'和'切换功能'选项

#### 2.4.5 答案之书模式
- 用户输入后直接进入答案之书界面
- 界面展示：'翻阅答案之书'按钮
- 交互：点击按钮随机显示指导性语句，提供'再次翻阅'选项

## 3. 设计风格

### 3.1 配色方案
以温暖的橙色为主色调，搭配清新的蓝绿色作为辅助色，营造轻松愉悦的决策氛围

### 3.2 视觉细节
采用圆润的卡片式设计，适度的阴影层次增强立体感，图标风格简洁现代，添加流畅的动画效果提升交互体验

### 3.3 整体布局
- 首页采用简洁的垂直布局，突出输入框和AI交互
- 功能页面采用居中对称布局，右侧预留聊天框空间
- 网站整体保持统一的视觉层次和交互逻辑
