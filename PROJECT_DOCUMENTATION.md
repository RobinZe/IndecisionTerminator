# 选择困难症终结者 - 项目文档

## 项目概述

选择困难症终结者是一个AI驱动的智能决策辅助网站，通过多种趣味化的工具和自然语言交互，帮助用户轻松解决各种选择困难问题。项目融合了随机选择算法、AI智能分析和现代UI设计，为用户提供直观、高效、有趣的决策体验。

## 核心功能

### 1. 掷硬币 🪙
- **功能描述**：为硬币的正反面分别指定事件，通过随机选择帮助做出二选一的决定
- **使用场景**：适合简单的二元决策
- **特色功能**：
  - 流畅的硬币翻转动画
  - 自定义正反面事件
  - AI自动识别和填充选项

### 2. 掷色子 🎲
- **功能描述**：支持输入2-6个选项，系统自动识别并为色子的6个面分配事件
- **使用场景**：适合多个选项的随机决策
- **特色功能**：
  - 智能解析输入（支持逗号、顿号、换行分隔）
  - 自动分配到6个面
  - 支持选项重复分配
  - 逼真的色子滚动动画

### 3. 概率转盘 🎡
- **功能描述**：自定义多个选项及其对应的概率权重，生成可视化转盘进行选择
- **使用场景**：适合需要考虑不同选项权重的复杂决策
- **特色功能**：
  - 动态添加/删除选项
  - 自定义每个选项的概率（支持归一化）
  - 精确的概率计算和指针定位
  - 流畅的转盘旋转动画
  - 视觉化概率分布

### 4. AI分析 ✨
- **功能描述**：基于文心大模型，深度分析选择困难的各个方面，提供理性的决策建议
- **使用场景**：适合需要深入分析的重要决策
- **特色功能**：
  - 智能识别选项和背景
  - 优劣势分析
  - 流式输出，实时查看分析过程
  - 客观中立的建议

### 5. 答案之书 📖
- **功能描述**：提供精心设计的指导性语句，为用户寻求灵感和启发
- **使用场景**：适合需要外部启发的决策时刻
- **特色功能**：
  - 50条精心设计的指导语句
  - 优雅的翻书动画效果
  - 神秘而有趣的体验

## AI智能特性

### 智能决策助手
项目集成了AI智能分析功能，能够：
- **自动工具选择**：根据用户输入的问题类型，自动选择最适合的决策工具
- **智能参数填充**：自动从用户描述中提取选项和参数
- **实时交互调整**：支持通过自然语言修改参数或切换工具
- **上下文理解**：理解复杂的决策场景和用户需求

### 工具选择规则
- **掷硬币**：恰好2个选项的简单决策
- **掷色子**：2-6个选项的决策
- **概率转盘**：需要考虑权重的多选项决策，或超过6个选项
- **AI分析**：需要深入分析优劣势的复杂决策
- **答案之书**：寻求灵感启发的决策

## 技术架构

### 前端技术栈
- **React 18 + TypeScript**：现代化的前端框架，提供类型安全和组件化开发
- **Tailwind CSS**：实用优先的CSS框架，实现快速响应式设计
- **shadcn/ui**：高质量的组件库，提供一致的设计语言
- **React Router**：单页应用路由管理，支持状态传递

### AI集成
- **文心大模型API**：提供强大的自然语言理解和分析能力
- **流式对话接口**：实时显示AI分析过程
- **智能解析**：自动提取选项、概率权重等关键信息

### 核心算法
- **公平随机算法**：确保所有随机工具的概率分布准确
- **精确角度计算**：概率转盘指针与结果100%一致
- **智能文本解析**：支持多种输入格式和分隔符

## 使用指南

### 快速开始流程
1. **首页智能分析**：描述选择困难，AI自动选择工具并填充参数
2. **直接选择工具**：点击首页的决策方式卡片，直接进入功能页面
3. **功能页面操作**：使用预填充参数或手动调整，执行决策
4. **实时调整**：通过底部输入框修改参数或切换工具

### 使用建议
- **简单决策**：使用掷硬币或掷色子
- **复杂决策**：使用概率转盘或AI分析
- **寻求灵感**：使用答案之书
- **重要决策**：结合AI分析和其他工具综合判断
