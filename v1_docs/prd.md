# 选择困难症终结者需求文档

## 1. 网站概述

### 1.1 网站名称
选择困难症终结者

### 1.2 网站描述
一个通过随机概率和智能分析帮助用户解决选择困难的决策辅助网站，提供多种趣味化的选择工具。

### 1.3 网站功能
- 掷硬币：为正反面指定事件，随机选择结果
- 掷色子：语音或文字输入事件，系统识别并随机选择
- 概率转盘：自定义条目和概率，生成转盘进行选择
- AI分析：智能分析各种情况，提供选择建议
- 答案之书：随机显示指导性语句帮助决策

## 2. 功能详细说明

### 2.1 主页面
展示五种选择模式的入口，用户可点击进入对应功能页面。

### 2.2 掷硬币模式
- 用户为硬币正面和反面分别输入对应的事件名称
- 点击'掷硬币'按钮执行随机选择
- 显示硬币翻转动画和最终结果

### 2.3 掷色子模式
- 支持文字输入或语音输入描述色子六个面对应的事件
- 系统自动识别并为每个面分配事件（允许重复）
- 点击'摇色子'按钮显示色子滚动动画和结果

### 2.4 概率转盘模式
- 用户添加多个选择条目并设置对应概率
- 系统根据概率生成可视化转盘
- 点击'开始转盘'后转盘旋转并停在选中区域

### 2.5 AI分析模式
- 用户通过语音输入描述选择问题
- AI分析各个选项的优劣势
- 提供理性的选择建议和参考依据

### 2.6 答案之书模式
- 页面显示提示文字：'想好一个问题并点击翻阅答案之书'
- 点击'翻阅答案之书'按钮随机显示指导性语句
- 提供'再次翻阅'选项供用户重新获取答案

## 3. 设计风格

### 3.1 配色方案
以温暖的橙色为主色调，搭配清新的蓝绿色作为辅助色，营造轻松愉悦的决策氛围。

### 3.2 视觉细节
采用圆润的卡片式设计，适度的阴影层次增强立体感，图标风格简洁现代，添加流畅的动画效果提升交互体验。

### 3.3 整体布局
主页采用网格式布局展示五种模式，各功能页面采用居中对称布局，确保操作便捷且视觉平衡。
