# 选择困难症终结者 - 功能说明

## 项目概述

选择困难症终结者是一个帮助用户解决选择困难的决策辅助网站，通过多种趣味化的工具，让决策变得简单有趣。

## 核心功能

### 1. 掷硬币 🪙
- **功能描述**：为硬币的正反面分别指定事件，通过随机选择帮助做出决定
- **使用场景**：适合二选一的简单决策
- **特色**：流畅的翻转动画效果

### 2. 掷色子 🎲
- **功能描述**：输入多个选项（用逗号、顿号或换行分隔），系统自动识别并为色子的6个面分配事件
- **使用场景**：适合2-6个选项的决策
- **特色**：智能识别输入，支持选项重复分配

### 3. 概率转盘 🎡
- **功能描述**：自定义多个选项及其对应的概率权重，生成可视化转盘进行选择
- **使用场景**：适合需要考虑不同选项权重的复杂决策
- **特色**：
  - 支持添加/删除选项
  - 自定义每个选项的概率
  - 动态转盘旋转动画

### 4. AI分析 ✨
- **功能描述**：描述你的选择困难，AI会分析各个选项的优劣势，提供理性的决策建议
- **使用场景**：适合需要深入分析的重要决策
- **特色**：
  - 基于文心大模型的智能分析
  - 流式输出，实时查看分析过程
  - 客观中立的建议

### 5. 答案之书 📖
- **功能描述**：在心中默念问题，翻阅答案之书获得指导性语句
- **使用场景**：适合寻求灵感和启发的决策
- **特色**：
  - 50条精心设计的指导语句
  - 优雅的翻书动画效果
  - 神秘而有趣的体验

## 设计特色

### 配色方案
- **主色调**：温暖的橙色（#FF7A3D）
- **辅助色**：清新的蓝绿色（#2EAAA0）
- **背景色**：柔和的浅色系
- **整体氛围**：轻松愉悦，减轻决策压力

### 交互设计
- 圆润的卡片式设计
- 流畅的动画过渡效果
- 清晰的视觉层次
- 响应式布局，适配各种设备

### AI集成
- 文心大模型API

- 随机算法实现公平选择

## 使用建议

1. **简单决策**：使用掷硬币或掷色子
2. **复杂决策**：使用概率转盘或AI分析
3. **寻求灵感**：使用答案之书
4. **重要决策**：结合AI分析和其他工具综合判断

## 注意事项

- AI分析仅供参考，最终决定权在用户
- 答案之书的语句具有启发性，不应作为唯一依据
- 所有随机结果均基于公平的随机算法
- 建议在使用工具后，结合自己的实际情况做出最终决定
